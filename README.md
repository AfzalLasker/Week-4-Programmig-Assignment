# Cricket Quiz 1.4
This is a C++ program that runs a short cricket quiz.  
It asks the user four cricket-related questions, checks the answers, and shows the final score and percentage.

## Features
- Asks 4 cricket questions  
- Validates user input  
- Counts correct answers  
- Shows score and percentage  
- Displays a menu using a switch statement  
- Uses loops (for-loop, while-loop, do-while loop)  
- Saves a score report to a text file
